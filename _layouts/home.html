Applications

J'adore bidouiller, concevoir et fabriquer des choses. Pour moi la programmation est donc un régal. Et puis autant que ce soit utiles ! Mes application sont donc toutes open source. C'est en particulier la programmation d'application web qui me fascine. Mais sont inconvennient majeur est qu'elle nécessite un serveur web pour heberger l'application. Github propose d'heberger le code source d'applciation sous la forme de dépot Git. Et permet également d'héberger des sites statiques.

C'est pourquoi j'ai décidé d'heberger mes applications sur Github Pages. Contrairement à un site dynamique, le site statique n'a pas reccours à du code coté serveur (par ex PHP) pour intéroger des bases de données et produire du contenu de manière dynamique. Cette logique ma donc contraint à développer des application web s'executtant sur le navigateur du client après chargement du nécessaire depuis le serveur... Bref... Je vous ai perdu (mais c'est déjà énorme que vous ayez lu jusque ici), ce sont des "Single Page Applications" développées en javascript et utilisant le Framework AngularJS.
